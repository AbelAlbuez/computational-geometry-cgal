BraTS glioma — interpolación lineal de contornos

Dashboard con métricas por par (Dice, IoU, Hausdorff, areaErr) comparando el contorno interpolado a t=0.5 contra el slice GT real intermedio. El visualizador interactivo permite recorrer t ∈ [0, 1] en pasos de 0.1; los contornos azul (A), rojo (B) y naranja (GT) permanecen fijos, y el contorno verde se actualiza con los vértices precalculados para cada t.

Verde: Dice ≥ 0.90 / IoU ≥ 0.80 / Hausdorff ≤ 3 / areaErr ≤ 0.05 · Amarillo: zona intermedia · Rojo: peor desempeño.
